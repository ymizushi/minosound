minosound
================================

# みのさうんど

================================

迷宮を自動生成して迷路を解いていくと音楽が奏でられるアプリです
2013/05/28 App Storeにてリリース.
現在機能追加・修正中です

# 実装済
- 迷路自動生成
- 最短経路探索
- なぞると探索済みマークをつける
- タイマー
- スタート・ゴール
- 最短経路で耳障りの良い音を自動生成してならす

# 未実装
- オンラインでランキング
- レベル機能
- インターフェースとかも格好いい感じにする
